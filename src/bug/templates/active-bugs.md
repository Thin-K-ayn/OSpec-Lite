# Active Bugs

This file tracks bug items that are still being worked through.

- Add a new item with `oslite bug new "<title>"`.
- Update the matching `bug-####` section in place.
- Run `oslite bug fix <bug-id> [path]` after the fix summary and touched files are real.
- Run `oslite bug apply <bug-id> [path]` after verification and durable lessons are real.
- Applied items are removed from this file after their knowledge is persisted.
